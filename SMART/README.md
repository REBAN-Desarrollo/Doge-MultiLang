## Como leer estos documentos

Cada documento sigue la estructura:

1. **Contexto** — Donde estamos hoy (datos reales)
2. **Objetivos SMART** — Macro a micro, con entregables concretos
3. **Entregables** — Que produce esta persona, con que calidad, para cuando
4. **Dependencias** — Que necesita de otros stakeholders
5. **Preguntas abiertas** — Lo que falta resolver
