# Libraries & SDKs Overview

ElevenLabs offers language-specific libraries across three categories:

## Official REST API Libraries

The company maintains supported libraries for Python and Javascript (Node), available through PyPI and npm respectively, with documentation on GitHub.

## Official Agents Platform Libraries

ElevenLabs provides native libraries for the Agents Platform spanning multiple ecosystems: "Javascript, React, React Native, Python, Swift, Kotlin, and Flutter." These are distributed through npm, PyPI, Maven, Github, and Pub package managers.

## Community-Maintained Options

The page notes that "These libraries are not officially supported by ElevenLabs, but are community-maintained." Three third-party integrations are listed:

- Vercel AI SDK (npm)
- .NET library via NuGet
- Unity integration through OpenUPM

Each entry includes links to documentation and respective package managers for installation.
