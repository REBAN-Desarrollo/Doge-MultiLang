# Subtitles

This documentation provides a comprehensive guide to ElevenLabs' Subtitle Editor tool.

## Main Features

The guide walks users through 10 key steps for subtitle editing:

1. **Access**: Navigate via the Speech to Text page and open any transcript
2. **Basic edits**: Rename subtitles using the right panel
3. **Adding subtitles**: Use the "+" button next to "Subtitles" or enable during transcript creation
4. **Formatting rules**: The editor provides real-time feedback using color coding for constraints like characters per line, lines on screen, and cue length
5. **Text editing**: WYSIWYG editor with undo functionality
6. **Timing adjustments**: Drag timeline handles or enter exact timestamps
7. **Cue management**: Split cues with Enter, merge using dedicated button
8. **Add/delete**: Create new cues or remove existing ones
9. **Playback control**: Adjust media playback speed
10. **Export**: Choose between SRT or VTT formats

## Key Distinctions

The FAQ clarifies that transcripts and subtitles are independent -- changes to one don't affect the other. Subtitles have specific formatting constraints that transcripts lack, including no word-level timestamps or speaker labels.
