# Transcript Editor

The Transcript Editor is a WYSIWYG tool within ElevenLabs' Speech to Text platform that allows users to comprehensively edit and manage transcripts.

## Core Editing Features

Users can modify transcript content by clicking anywhere and typing, with undo functionality via Command+Z. The interface supports renaming transcripts through a right-side panel and enables granular control over segment timing through draggable timeline handles or manual timestamp entry.

## Segment Management

The editor permits users to split a segment by clicking in the text where you want to split and pressing **Enter**. Merging requires two conditions: segments must belong to the same speaker and be adjacent. Users can add new segments via an 'Add Segment' icon or remove them using a Delete button.

## Speaker Operations

Multiple speaker management options include:

- Individual segment reassignment via dropdown menus
- Bulk reassignment across all segments from one speaker
- Ability to add or delete speaker tracks

The system warns that deleting a speaker removes all associated segments. Users can reorder speaker tracks and customize their colors through an orb interface.

## Export and Additional Features

The platform supports exporting in five formats:

- Plain text
- JSON
- HTML
- SRT
- VTT

Users can adjust playback speed and add subtitles. ElevenLabs also offers professional human transcription services starting at $2.00 per minute through their Productions service.
