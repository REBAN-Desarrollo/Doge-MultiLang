# Usage Analytics

This documentation explains how to access and utilize the Developers page for monitoring platform activity across ElevenAgents and ElevenCreative.

## Access Point

The Developers page is located at the bottom of the sidebar.

## Main Features

The page contains three primary tabs:

### 1. Usage Tab

Displays customizable metrics including credits, with breakdown options by voice, product, or API key. For workspace members, additional filtering by individual user or group is available. Data can be viewed across different time intervals (daily, weekly, monthly, or cumulative).

### 2. Analytics Tab

Provides detailed insights into workspace API requests and webhooks with filtering capabilities across multiple time periods and granularity levels, plus success rate and latency monitoring.

### 3. Request Log

Enables viewing and filtering of specific API requests for debugging and monitoring functions.

## Export Functionality

Users can download current data views as CSV files using the "Export as CSV" button, making it easy to perform external analysis or record-keeping.

This suite of tools is designed to help users understand their own usage patterns or, for those who've integrated the platform into their products, analyze customer usage data.
